# Growtex 🚀

Growtex is a modern business landing page built to help businesses expand globally and connect with international opportunities.

The project is designed with a responsive UI, smooth user experience, and supports both **light and dark themes**.

## 🌐 Live Website

https://www.growtex.in

## ✨ Features

* Responsive landing page design
* Light and dark theme support
* SEO-friendly structure
* Fast deployment with Vercel
* Custom domain integration with GoDaddy
* Sitemap support for search engine indexing
* Modern UI/UX design

## 🛠️ Tech Stack

* React
* Vite
* JavaScript
* CSS / Tailwind CSS
* Vercel Deployment

## 📂 Project Structure

```bash
src/
public/
README.md
package.json
```

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/growtex332/Growtex.git
```

Move into the project directory:

```bash
cd Growtex
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

## 🌍 Deployment

The project is deployed on **Vercel** with custom domain support.
